# LeetCode
## 1.时间复杂度
1.二分查找的时间复杂度是O(logN)  
2.插入排序的最差时间复杂度是O(N2)  
3.冒泡排序的时间复杂度是O(N2)  
4.归并排序时间复杂度O(N*logN) 空间复杂度O(N)  
5.快速排序的时间复杂度是O(N*logN) 最差空间复杂度O(N)

## 2.一些数学运算
1.异或(^)  
  0^1=1; 0^0=0; 1^1=0 0^N=N;  
2.与(&)  
0&1=0; 0&0=0; 1&1=1 见0则为0,两个同时为1则为1  
3.或(|)  
0|0=0; 0|1=1; 1|1=1 见1为1，两个同时为0则为0  
## 3.求中点

"<<"为左运算符，num<<1,相当于num乘以2
